📦 JavaScript Variables & Data Types

1. What is a Variable?

A variable is like a box that stores a value. You give it a name and put data inside.

let name = "Ravi";
let age = 20;
console.log(name);
console.log(age);

2. var, let, and const

var - old way, can be re-declared
let - modern, value can change
const - value cannot change 

var city = "Delhi";
let score = 50;
const country = "India";

score = 75;  // ✅ allowed
// country = "USA";  ❌ not allowed

console.log(city);
console.log(score);
console.log(country);

3. Naming Rules

Rules for naming variables in JavaScript:

// ✅ Valid names
let firstName = "Sita";
let _age = 22;
let $price = 100;

// ❌ Invalid names
// let 1name = "bad";   → starts with number
// let my-var = 5;      → hyphen not allowed
// let let = 10;        → reserved keyword

console.log(firstName);
console.log(_age);
console.log($price);

4. Scope

Scope means where a variable can be used. Variables inside a function are not available outside.

let globalVar = "I am global";

function test() {
  let localVar = "I am local";
  console.log(globalVar);  // ✅ works
  console.log(localVar);   // ✅ works
}

test();
// console.log(localVar); ❌ error outside

5. Reassignment

You can change the value of a let variable anytime.

let marks = 60;
console.log("Before: " + marks);

marks = 85;
console.log("After: " + marks);

6. Data Types

Data type tells what kind of value is stored in a variable.

let num = 10;          // Number
let name = "Amit";     // String
let isPass = true;     // Boolean
let x = null;          // Null
let y;                 // Undefined

console.log(typeof num);
console.log(typeof name);
console.log(typeof isPass);
console.log(typeof x);
console.log(typeof y);

7. Primitive Data Types

These are the basic/simple data types in JavaScript.

let num = 42;           // Number
let name = "Priya";     // String
let isTrue = true;      // Boolean
let nothing = null;     // Null
let notDefined;         // Undefined

console.log("Number: " + num);
console.log("String: " + name);
console.log("Boolean: " + isTrue);
console.log("Null: " + nothing);
console.log("Undefined: " + notDefined);

8. Non-Primitive Data Types

These store multiple values. Examples: Array and Object.

// Array - list of values
let fruits = ["Apple", "Mango", "Banana"];
console.log(fruits[0]);
console.log(fruits[1]);

// Object - key and value pairs
let student = { name: "Raj", age: 18 };
console.log(student.name);
console.log(student.age);

9. Type Checking

Use typeof to check what type a variable is.

let a = 100;
let b = "Hello";
let c = true;
let d = [1, 2, 3];
let e = { name: "Tom" };

console.log(typeof a);  // number
console.log(typeof b);  // string
console.log(typeof c);  // boolean
console.log(typeof d);  // object
console.log(typeof e);  // object

10. Type Conversion

You can convert one type to another using built-in functions.

// String to Number
let str = "50";
let num = Number(str);
console.log(num + 10);       // 60

// Number to String
let n = 100;
let s = String(n);
console.log(s + " rupees");  // 100 rupees

// Number to Boolean
console.log(Boolean(1));     // true
console.log(Boolean(0));     // false
